## Assignment 2

1. Go to folder assignment-2

2. Install the dependencies:

```bash
npm install --legacy-peer-deps
```

3. Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
